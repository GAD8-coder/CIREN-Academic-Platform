# CIREN Academic Platform

A collaborative academic study platform built for students, by students — powered by the CIREN club.

-----

## 🎯 About

CIREN Academic Platform is a study tool designed to support students with resources, collaboration, and AI-assisted learning — similar to platforms like Thea AI but tailored specifically for academic use within our community.

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/GAD8-coder/CIREN-Academic-Platform.git
cd CIREN-Academic-Platform
```

### Backend Setup

```bash
cd backend
python -m venv .venv
.venv\Scripts\activate      # Windows
pip install -r requirements.txt
flask run
```

### Frontend Setup

```bash
cd frontend
npm install
npm start
```
